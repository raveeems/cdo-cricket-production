Fix the impact bench ACTIVE badge showing incorrectly for all impact candidates.

The bug: isImpactPlayer=true is set for all admin-designated candidates.
The frontend reads isImpactPlayer to show ACTIVE badge — so all candidates 
show ACTIVE even when no impact sub has been used yet.

The fix: add isImpactActivated field (driven by officialImpactSubUsed) 
to the bench player shape, and use that for the ACTIVE badge instead.

Do not break any existing functionality.
Push to main on https://github.com/raveeems/cdo-cricket-production when done.
