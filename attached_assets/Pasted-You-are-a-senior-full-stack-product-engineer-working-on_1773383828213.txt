A) TEAM STRUCTURE

Each fantasy entry must now include:

- 11 Main XI players
- 1 Primary Impact Pick
- 1 Backup Impact Pick

Important:
- Primary Impact Pick and Backup Impact Pick must be from the SAME real-world franchise/team
- Primary and Backup must be different players
- These are special conditional scoring picks
- They are NOT a normal free 12th and 13th player advantage

Validation:
- exactly 11 valid Main XI players
- exactly 1 Primary Impact Pick
- exactly 1 Backup Impact Pick
- Primary and Backup cannot be same player
- Primary and Backup must belong to same franchise
- preserve existing player-composition constraints unless they conflict with this brief

B) IMPACT SLOT RULE

The app must treat the Impact feature as one special Impact Slot with:
- Primary Impact Pick
- Backup Impact Pick

Scoring resolution order:
1. Check if Primary Impact Pick officially entered the match as the Impact Substitute
2. If yes, Primary becomes the active Impact Slot player
3. If no, check if Backup Impact Pick officially entered the match as the Impact Substitute
4. If yes, Backup becomes the active Impact Slot player
5. If neither officially entered as Impact Substitute, the Impact Slot gets 0 points

Very important:
- Being in the squad is not enough
- Being in the official Impact Player list is not enough
- Starting in the Playing XI is NOT enough for Impact Slot points
- Only official Impact Substitute entry activates the Impact Slot

Therefore:
- If Primary starts in the Playing XI, he gets 0 in the Impact Slot
- If Backup starts in the Playing XI, he gets 0 in the Impact Slot
- If Primary is in impact list but never enters, 0
- If Backup is in impact list but never enters, 0
- If Primary officially enters as Impact Substitute, Primary scores
- If Primary does not enter and Backup officially enters as Impact Substitute, Backup scores
- Never award points to both Primary and Backup in the Impact Slot

C) IMPACT SLOT POINTS

If the active Impact Slot player officially enters as Impact Substitute:
- award +4 playing points
- award all normal batting/bowling/fielding fantasy points based on actual contribution

If no active Impact Slot player:
- Impact Slot = 0 points

D) CAPTAIN / VICE CAPTAIN LOGIC FOR IMPACT SLOT

Users may assign Captain or Vice Captain to the Impact Slot.

Important final logic:
Captain/VC assignment must be tied to the Impact Slot, not permanently tied to only the Primary player name.

This means:
- If Captain is assigned to the Impact Slot:
  - if Primary becomes active, Captain bonus applies to Primary
  - else if Backup becomes active, Captain bonus applies to Backup
  - else if neither becomes active, Captain bonus for Impact Slot = 0
- Same logic for Vice Captain

Playing bonus remains:
- Normal player: +4
- Captain: +8
- Vice Captain: +6

Important:
- No auto reassignment of Captain/VC to any player outside the Impact Slot
- If neither Primary nor Backup becomes active, the Impact Slot gets 0 and no Captain/VC benefit applies there

If current app already has Captain/VC multipliers for total performance points, preserve current multiplier behavior unless it conflicts with this brief. Only ensure the Impact Slot activation controls whether Captain/VC applies there.

E) MAIN XI SCORING

The 11 Main XI players continue to score under the current fantasy scoring system.
Do not break or unnecessarily change the existing scoring logic for the Main XI.

F) WINNING TEAM PREDICTION

Each team entry must include a winning team prediction.

Rules:
- Correct winner prediction = +50 points
- Wrong prediction = 0

Tie / Super Over:
- use official match winner for +50

No Result / Abandoned / Washed Out:
- contest becomes VOID
- do not calculate final contest standings
- do not include the contest in totals / leaderboard aggregation / settlement
- no winner bonus

G) WEEKLY MULTI-TEAM RESTRICTION

Final rule:
A user can create 2 or 3 teams in a match only 3 times in a week.

For all other matches that week:
- user can create only 1 team

Week definition:
- Monday 00:00:00 IST to Sunday 23:59:59 IST

Important counting rule:
Count weekly multi-team usage only based on FINAL submitted teams at MATCH LOCK TIME.

Examples:
- user creates 2 drafts, deletes one before lock, final locked count = 1 → do not consume usage
- user locks 2 teams for a match → consume 1 weekly usage
- user locks 3 teams for a match → consume 1 weekly usage
- user locks 1 team → consume 0 usage

H) INVISIBLE MODE

Users may activate Invisible Mode for 1 match per week.

Week definition:
- Monday to Sunday IST

If Invisible Mode is enabled for a match:
- it applies to ALL teams that user created for that match

Before match completion / before final standings:
Other users must NOT see:
- full team
- captain
- vice captain
- Primary Impact Pick
- Backup Impact Pick
- winning team prediction
- live points
- live rank
- scoring breakdown
- any indirect stat that leaks composition

Other users should only see:
- that the user has joined
- number of teams entered

The user himself should still be able to see:
- his own teams
- own captain / vice captain
- own Impact picks
- own winner prediction
- own live points
- own own standings information

After match completion and final standings calculation:
- reveal Invisible entries normally

Rules:
- user can turn Invisible Mode on/off before lock
- after lock it is frozen
- max 1 Invisible Mode match per IST week
- resets every Monday IST

I) ADMIN PRE-MATCH STATUS SETUP

Admin must be able to mark for each real team in a match:
- Playing XI
